# Shiftendo

**Shiftendo** is a 2D, top-down game, about a boy who is working on a ghost infested graveyard to save up money for Nintendo Switch 2.

---

## Used technologies

- Unity (2022.3.47f1)
- C# Programming language
- Aseprite - For 2D sprites
- sfxr - For game sounds
